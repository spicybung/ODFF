# ODFF
*ODFF* is an open source model conversion program for SAMP. That is, it will attach the proper SAMP collision to the object of your choice, allowing it to display in SAMP/OMP.


### How It Works
The way it works is loosely based on how KDFF GUI operates, though features are intended to be added.
