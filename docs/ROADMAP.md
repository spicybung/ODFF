# ODFF Roadmap

## 0.1

- Single and group DFF workflow
- OpenGL viewport
- Generic PC DFF parsing
- TXD metadata parsing
- Collision generation and attachment state
- ODFF collision project persistence

## 0.2

- D3D8 and D3D9 native raster decoding
- DXT1, DXT3, and DXT5 decompression
- Material-to-TXD texture binding
- Alpha sorting and alpha test controls
- Per-material visibility
- Normal generation for geometry without normals

## 0.3

- Validated GTA COL1, COL2, COL3, and SA-MP COL export
- Collision surface/material editor
- Shadow mesh and face-group controls
- Batch collision naming and export
- Existing embedded collision import

## 0.4

- PS2 native geometry unpacking
- Xbox native geometry unpacking
- Skin and HAnim visualization
- 2DFX inspection
- BinMesh strip/list handling
- MatFX rendering

## 1.0

- DFF writing
- TXD writing and texture replacement
- Embedded collision attachment directly into DFF
- Stable command-line batch mode
- Regression corpus and format tests
